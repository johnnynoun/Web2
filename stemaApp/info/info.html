About Stema

Stema is a web appliation with the goal of giving the user nutritive information about a certain product(ex: fruits, vegetables etc.), the user will be able to search for the product know info about this product.

How to use Stema

Stema is really easy to use! All the user needs to do is go to home and search the product they want to see, also if the user needs more info about the products they can go in other categories like Nutritional Facts or Allergies to get more detail about the product searched.
